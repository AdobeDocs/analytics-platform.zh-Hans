# 执行基本分析

创建连接和数据视图后，利用 Analysis Workspace 的强大功能和灵活性分析导入的数据。您可以随意尝试和拖动维度和量度，更改维度和量度归因设置、友好名称、时区、会话设置等。

以下是 Workspace 中基础的可视化图表示例。例如，您可以：

* 创建排名报表，其中数据源显示排名最靠前的事件、会话和人员。

* 创建在线商店收入与实体店收入的趋势报表，比较两个数据源随时间推移发生的变化。

# 项目符号图表

让您可以查看您感兴趣的值与其他性能范围（目标）进行比较或衡量的结果。

![显示“图表类型”选项、“项目符号选项”以及其他图表设置的“可视化设置”窗口。](assets/bullet-image.png)

项目符号图表具有一个主要的测量方法（例如，当前年初至今的收入），允许您输入定性范围、绩效范围（例如，与目标收入相比）。您可以指定高、中和低。您可以在[!UICONTROL 可视化设置]中指定目标范围。

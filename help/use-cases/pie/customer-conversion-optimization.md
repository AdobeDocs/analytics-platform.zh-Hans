# 客户转化优化

品牌通常将总体转化率视为成功的滞后指标，并专注于转化率优化以增加展示次数。 但转化率是所有虚指标中的王；它可能表示短期的胜利，但通过将数量置于客户体验质量之上，您可能会对品牌造成长期损害。

## 用例概述

客户转化优化用例的主要目标是推进当前处于考虑阶段的前景成为转化客户。 要实现这一点，您需要考虑每个客户的转化需求，而不是只查看汇总转化率。

### 和老家伙一起出去

许多组织在了解宏观层面的转化时（例如创收事件），如果与任何类型的转化事件的成因相结合，将不断面临挑战。 对于微观和纳米客户而言，很难针对不同的同类群组启动有针对性的促销活动来推动注册和转化。

典型的转化优化策略还被单独限制在数字渠道，例如Web或移动设备。 由于缺乏在其他接触点上的客户互动知识，品牌很难规划有影响力的多渠道体验，以指导人员从考虑因素到转化。 因此，它们可能会继续提供无关紧要或陈旧的信息传递，进一步冒着品牌长期受损的风险。

### 使用新的

通过客户转化优化，品牌可以将当前处于考虑阶段的目标客户转化为已转化的客户。 互联转化优化工具一对一地分析每个潜在客户的体验，并相应地改进其独特体验。 通过关注每个用户的历程并专注于单个客户的成功，您可以：

* 通过特征和行为数据分析令人兴奋体验的所有上下文驱动因素，以改进转化事件。
* 对尚未注册或转换的假名配置文件使用高级分段和激活工具。
* 使用兴趣和首选渠道创建个性化设置，从而在广告、社交、电子邮件、文本等多个领域一对一扩展体验。
* 使用AI建模来识别指标并将用户路由到最佳路径，例如免费试用或聊天。

客户转化优化关乎平衡。 短期成功指标（如转化率）很重要，但长期指标（如忠诚度、终生价值、品牌情绪和净推广者得分）是确保品牌有一个光明和可持续的未来的因素。

因此，不要试图从广泛的细分市场产生相同的转化事件，而要专注于建立持久的一对一客户关系，并分析成功的信号和模式。 作为以客户体验为中心的战略的副产品，您的转化率将自然而然地提高。

### Journey Optimizer

Journey Optimizer可帮助您编排最佳的1:1后续步骤，将客户从考虑转移到购买。 在单个历程画布中，您构建一个渐进式历程，根据客户的实时信号触发下一个最佳操作。 这些操作可以包括带有个性化优惠和产品推荐的消息，以将客户从考虑推向购买。

# 将B2B数据摄取到AEP并在CJA中报告

利用AEP上新提供的B2B数据集，为B2B营销人员提供有价值的分析和报告解决方案。 添加功能板，使客户能够将流经AEP的B2B数据可视化，以提高透明度，从而将这些功能板添加到为AEP用例中的B2B提供支持的数据中。

# 日期范围概述

日期范围是一种可在Workspace项目中使用的组件类型。 它们允许您查看随时间推移的数据趋势，或查看事件的最大发生时间。 日期范围采用颜色编码的紫色。

自定义日期范围允许您自定义在Workspace项目中看到的日期。 使用日期范围管理器 [管理](manage.md)[或创](create.md) 建自定义日期范围。

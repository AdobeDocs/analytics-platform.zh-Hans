# 使用函数

您可以使用函数过滤、排序数据，以及进行简单和复杂的统计分析。

有关所有函数的列表，请参阅[基本函数](/help/components/calc-metrics/cm-functions.md)和[高级函数](/help/components/calc-metrics/cm-adv-functions.md)。



>[!NOTE]
>
>计算量度生成器的定义中包含函数时，应始终在拖入量度或过滤器之前应用该函数。
>
